Orion Viewer Privacy Policy

Orion Viewer application for Android platform use only one permission that can be considered as dangerous - MANAGE_EXTERNAL_STORAGE. Orion viewer doesn't collect any user data.

List of accessing permissions:

  • READ_EXTERNAL_STORAGE - to access book files
  • WRITE_EXTERNAL_STORAGE - to save application and book setting, export and save bookmarks
  • WAKE_LOCK - to keep device in switched on state
  • MANAGE_EXTERNAL_STORAGE - to access folders for Orion's file manager
